# Meeting Prep AI Assistant

A context-aware AI strategist that helps you prepare for meetings by generating smart questions, identifying risks, and defining critical decision points.

###  Key Features
- **Project Memory:** Remembers past meeting outcomes to improve future advice.
- **Strategic Briefs:** Generates 5 Smart Questions, 3 Likely Risks, and 2 Decision Points.
- **Powered by:** Streamlit & Gemini 2.5 Flash.
